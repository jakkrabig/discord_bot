# discord_bot

>Sample discord bot

## Installation

- Clone repo

```shell
$ git clone https://github.com/jakkrabig/discord_bot.git
```


- Change directory

```shell
$ cd discord_bot
```

- Install dependencies

```shell
$ npm install
```

- Create file `.env` and declare variable below

```
TOKEN=YOUR_BOT_TOKEN
PREFIX=YOUR_COMMAND_PREFIX
```

- Bot log in
```shell
$ node index.js
```
